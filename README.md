# Vueテンプレート

create vueで作ったプロジェクトを使いやすいようにカスタマイズしたもの。

## Version

Voltaで管理しています。
- Node 20.10.0
- npm 10.2.3

## start
1.  **install**  

    クローンした後に実行してください。

    ```
    npm i
    ```

2.  **develop**

    ```
    npm run dev
    ```

3.  **build**

    ```
    npm run build
    ```

## format
```
npm run format:all
```
